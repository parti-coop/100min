참가신청

100년 토론광장의 국민 참여단 신청안내입니다

100년 토론광장은 3‧1운동과 대한민국임시정부 수립과
지난 100년의 역사에서 우리가 계승할 가치를 찾아보고,
앞으로 우리가 어떤 세상을 만들어야 하는지를 함께 토론하는 국민참여 행사입니다.

토론 결과는 참가자 명의의 선언서/제안서로 만듭니다.

신청시 주의사항

  • 100년 토론광장은 이 나라 주권자인 우리가 직접 100주년의 의미를 토론하고, 대한민국의 새로운 모습을 그려보는 국민주도의 토론광장입니다.
  • 참여자는 신청자 중 나이, 성별을 고려하여 무작위로 선발합니다.
  • 선발 통보 후 무응답 시 불참의사로 간주합니다.
  • 소정의 참가비가 지급되므로 사전신청자는 참석이 어려울 경우 사전에 반드시 불참 통지를 해주시기 바랍니다.
  • 거주 지역과 상관없이 원하는 지역에 자유롭게 신청 가능합니다.
  • 토론의 원활한 진행을 위하여 참가자들은 홈페이지에 게시된 사전자료집을 반드시 읽고 오시기 바랍니다.

신청 지역

  • 영남권 ( 2019.5.31(금) 오후 2시 / 경남도청 대회의실 본관(창원) )
  • 호남•제주권 ( 2019.6.5(수) 오후 2시 / 광주 국립아시아문화전당 컨퍼런스홀 )
  • 충청권 ( 2019.6.10(월) 오후 2시 / 대전 컨벤션센터 그랜드볼룸 )
  • 강원권 ( 2019.6.28(금) 오후 2시 / 춘천세종호텔 사파이어홀(춘천) )
  • 수도권 ( 2019.7.13(토) 오후 2시 / 서울 세종문화회관 세종홀 )
